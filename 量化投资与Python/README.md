## Numpy模块介绍
### Numpy简介
Numpy是高性能科学计算和数据分析的基础包。他是pandas等其他各种工具的基础。

Numpy的主要功能：
    - ndarray，一个多维数组结构，高效且节省空间
    - 无需循环对整组数据进行快速运算的数学函数
    - 线性代数、随机数生成和傅里叶变换功能
